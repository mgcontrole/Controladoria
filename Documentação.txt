Documentação de estrutura base dos projetos.

Pasta Raiz/ 


* Sempre divida o js/css/html em arquivos diferentes
(Colocar os arquivos '.js' e '.css' dentro da pasta 'static')
 - static/arquivo.javascript
 - static/arquivo.css

(Todos os arquivos usados para base de dados devem ficar dentro de uma pasta 'data'. OBS: Nunca salvar os arquivos diretamente aqui dentro, use subpastas)
 - data/ 

(Mantenha o arquivo '.html' sempre dentro da pasta raiz e com o nome "index.html")
 - index.html

(Para arquivos que compõe o backend ex: 'Python', 'Node'. Use a pasta backend)
 - backend/

(Os arquivos que não fazem parte da montagem da interface, deixe fora do diretório usando o .gitignore)
